Atividade 2


Passos e comandos necessários para uso do github.

mkdir para criação de uma pasta.
cd para deslocar para dentro dela.
git init para iniciar um novo repositório.
git add para indexar um novo arquivo.
git status para ver a situação dos arquivos.
git commit -m para validar alterações e enviar ao HEAD, além de usado para comentar a alteração.
git log para verificar os commit feitos.
git remote add origin +https para conectar o repositório local ao remoto.
git push origin master	envio do HEAD ao repositório remoto.
